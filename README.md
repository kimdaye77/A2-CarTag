## 주요 기능
### 예산 설정
- 드래그를 통해 화면 내 이동 가능
- 선택한 옵션 가격이 반영되어 설정한 예산이 얼만큼 남은지 보여줍니다.
<img width="500" src="https://github.com/softeerbootcamp-2nd/A2-CarTag/assets/63107805/021d8943-c2d2-4f1e-938d-db65ec39d266"/>


### 360도 회전
- 드래그를 통해 선택한 차량 외부 360도 회전 가능
<img width="500" src="https://github.com/softeerbootcamp-2nd/A2-CarTag/assets/63107805/82e611aa-2ea6-4029-af30-91a0c83a5910"/>

### 옵션 검색 자동완성
- 옵션명 또는 해시태그를 통해 옵션 검색 가능
- 키 이벤트로 검색 가능
<img width="500" src="https://github.com/softeerbootcamp-2nd/A2-CarTag/assets/63107805/08ad0105-25ed-42ff-8f35-402ee1bbf33a"/>


### 유사 견적
- 판매량 및 내 견적과 선택 옵션 유사도가 높은 다른 사람들의 실제 출고 견적을 보여줍니다.
<img width="500" alt="image" src="https://github.com/softeerbootcamp-2nd/A2-CarTag/assets/63107805/3f038d2e-09c9-4336-b7c7-9d391ff07366">

### 공유하기 기능
- 견적 완료 페이지에서 공유하기를 누르면 링크 복사
- 복사된 링크를 통해 선택한 차량 공유 가능

<img width="500" alt="image" src="https://github.com/softeerbootcamp-2nd/A2-CarTag/assets/63107805/32cb6a45-a41f-41a7-ad8d-453c370ce118"/>


### PDF 다운로드 기능
- 견적 완료 페이지에서 PDF 다운로드 가능
<img width="500" alt="image" src="https://github.com/softeerbootcamp-2nd/A2-CarTag/assets/63107805/516df2ef-efc8-466d-b0a4-e95a71149f7f"/>
